從電信業到資安外商：一場關於「純軟體」的震撼教育

發表於 2026-02-13 分類於 Diary

在趨勢科技待了快三個月，這段時間最深刻的體悟就是——我正在經歷一場全方位的「砍掉重練」。

開頭先自我聲明：為了保護讀者，這篇文章會先封印我在外商環境被植入的「晶晶體」。我希望這個網誌能是一個自由的空間，不管是技術、職場牢騷還是中英混雜的心情，都隨當下的狀態而定，看我當下想不想練習一下英文。

之所以重啟這個網站，是受到了我的 Mentor Yu-Jun 的啟發。看著他持續經營自己的網站，我才意識到對於工程師來說，這不僅是筆記，更是一種個人紀錄與專業的「數位資產累積」。

1. 關於「自律」的文化衝擊

外面常開玩笑說趨勢是假外商，但對比我待過的傳統電信業，這裡的文化衝擊依然巨大。

在前公司，即便努力朝向「科技電信業」轉型，但在資源分配與體制下，難免讓人感到疲憊。來到這裡後，我見識到了什麼叫真正的「成果導向」。

在這裡，可以看到一個能力好的員工 10 點多才上班，中午吃飯泡咖啡聊天搞了兩三個小時，下午 4、5 點跑去烤麵包或烤地瓜也沒人會管你，同事們還會很開心的加入。只要你能如期、高品質地交付工作，這些「自由」都是被尊重的。這也意味著，你需要極高的自律、自我技能提升，以及不斷學習的狀態。

2. PR 的三十道陰影：什麼叫嚴謹？

技術面是我挫折感最大的來源。以前覺得專案管理（Jira + Bitbucket）已經行之有年，但在純軟體產業面前，那真的差太多了。

從 SPEC 的定義、詳盡的 POC 到 Design Review，每一階段都是靈魂拷問。最震撼的莫過於 Code Review。以前發 PR 可能只是走個過場，但在這裡，發 PR 是一件嚴肅到近乎神聖的事情。從程式架構、效能優化到甚至是你的 PR Message，任何細節都會被同事刁。

曾有一次 PR 被留了 30 個 Comments，那種挫折感讓我深刻意識到：過去的任何經驗在這一刻並無幫助。我現在需要專注衝刺的，是學好 Golang、學會如何做 Design Review、學會如何做出一場好的 Presentation。

3. 另一個巨大的衝擊是「人」

在會議中，聽著同事用流利的英文與外國團隊溝通，看著周遭超過一半的人不是台大畢業就是留美歸國。對比自己所謂的「台灣野雞大學」背景，那種實力與學歷的差距是實打實的。

這認真啟發了我近期加強英文的決心，我已經開始在尋找美語補習班與線上課程，不想在未來的協作中，因為語言限制了技術的發揮。

4. 成長

最近我也在協助專案導入 datamodel-code-generator。這是一個能透過 Data Model 自動生成 OpenAPI (Swagger) YAML Spec 的工具，能極大地減少手動同步規格的錯誤，之後我會專門寫一篇文章談談細節。

我希望在這裡能越挫越勇。雖然現在 PR 被留 30 個留言，但我相信，總有一天我能挑戰「一次發 PR 就過」。到那天，我一定會再發一篇文章慶祝。

這是一個紀錄我如何從零開始，在資安叢林裡生存並成長的起點。
